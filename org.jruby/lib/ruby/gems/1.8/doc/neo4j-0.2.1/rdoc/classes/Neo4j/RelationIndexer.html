Class Neo4j::RelationIndexer
In: lib/neo4j/indexer.rb
Parent: Object

If node class A has a relation with type ‘d’ to node class B 

  A.x -d-> B.y  A.index d.y

If property y on a B node changes then all its nodes in the relation ‘d’ will be reindexed. Those nodes (which will be of type node class A) will use the same RelationIndexer to update the index document with key field ‘d.y’ and values of property y of all nodes in the relationship ‘d‘
